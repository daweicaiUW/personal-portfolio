# Dawei Cai Portfolio

A bilingual academic and engineering portfolio built with Next.js, TypeScript, Tailwind CSS, React components, Lucide icons, and editable static data files.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Build

```bash
pnpm build
```

## Update content

Profile, projects, research, experience, skills, documents, and translations live in `src/data/`.

## Replace assets

Add files under `public/media/` and `public/documents/` using the paths documented in those folders. Missing assets show clean placeholders instead of breaking the layout.
