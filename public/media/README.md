Place replaceable portfolio media in this folder.

- Profile photo: `public/media/profile/profile.jpg`
- Optional school emblem: `public/media/profile/school-emblem.png`
- UW-style background watermark: `public/media/backgrounds/uw-watermark.png`
- Project images: `public/media/projects/<project-slug>/cover.jpg` and gallery images
- Project videos: `public/media/videos/<name>.mp4`
- Research images and framework diagrams: `public/media/research/`
- Experience media: `public/media/experience/`

The UI includes graceful fallbacks, so missing files will not break page layout.
